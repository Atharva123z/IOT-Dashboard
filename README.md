# IOT-Dashboard
An IoT-based dashboard that collects real-time sensor data from ESP devices and displays it on a web interface using Firebase. The system enables seamless data transmission, cloud storage, and live monitoring through a dashboard.
# IoT Device Monitoring Dashboard 📊

A real-time dashboard to monitor sensor data from IoT devices using ESP32 and Firebase.

---

## 🚀 Features

- Real-time data visualization  
- Multiple sensor integration  
- Web-based dashboard  

---

## 🛠️ Tech Stack

- ESP32  
- Sensors: Ultrasonic, Pulse, Humidity  
- Firebase  
- HTML, CSS, JavaScript  

---

## ⚙️ Working

Sensors → ESP32 → Firebase → Web Dashboard
